# Chrome Tabs to Markdown

Copy all tabs in the current window as a Markdown list.

## Install (local)

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click "Load unpacked".
4. Select this folder.

## Usage

1. Click the extension icon.
2. Click "Copy Tabs".
3. The clipboard will contain:

```
- [ページタイトル](URL)
- [ページタイトル](URL)
```
